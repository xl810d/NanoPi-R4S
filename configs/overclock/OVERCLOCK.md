# NanoPi R4S RK3399 Overclocking

## 2.2GHz/1.8GHz
```
wget https://raw.githubusercontent.com/LewiVir/NanoPi-R4S/main/configs/overclock/rockchip-2.2_1.8.dtb
cp -f rockchip-2.2_1.8.dtb /mnt/mmcblk0p1/rockchip.dtb
reboot
```
